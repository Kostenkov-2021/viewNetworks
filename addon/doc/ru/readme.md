# View Networks (Просмотр сетей)

* Автор: Edilberto Fonseca <edilberto.fonseca@outlook.com>.
* Дата создания: 07/08/2022.

----------

Это дополнение выводит список всех сетей, сохранённых на вашем устройстве. Оно также пытается, когда это возможно, восстановить пароль от них.

**Примечание:** Это дополнение не взламывает и не пытается взломать пароль Wi-Fi. Всё, что оно делает или пытается сделать — это получить пароль, сохранённый в системе при первом подключении к сети.

## Как использовать

Использование дополнения относительно простое. Вызовите его с помощью горячих клавиш или через меню NVDA Сервис.

### Диалог «Список сетей»

В этом диалоговом окне перечислены все Wi-Fi сети, сохранённые в системе. Имеется только одна кнопка «Закрыть». Для выхода из окна можно также нажать клавишу `Escape` или сочетание клавиш `Alt+C`.

### Диалог «Восстановление пароля»

В этом диалоговом окне есть поле для поиска, куда нужно ввести имя сети, для которой планируется попытаться восстановить пароль. Как только будет нажата кнопка «Поиск» (`Alt+P`), содержимое ключа (пароля) отобразится в соответствующем поле. В противном случае появится сообщение с предупреждением, что получить информацию о сети «[имя сети]» не удалось.

В этом диалоговом окне также есть ещё две кнопки:
*   **Очистить** (`Alt+L`) — очищает поля и помещает фокус обратно в поле поиска.
*   **Закрыть** (`Escape` или `Alt+F`) — завершает работу диалогового окна.

## Сочетания клавиш

1.  `Windows+Alt+N`: Открывает диалог сохранённых сетей.
2.  `Windows+Alt+O`: Показывает информацию о дополнении.
3.  `Windows+Alt+P`: Открывает диалог «Восстановление пароля».
4.  `Windows+Alt+J`: Открывает справку дополнения.

## О дополнении

Кнопка «О программе» («About») отображает сообщение со следующей информацией о дополнении:

*   Версия;
*   Автор;
*   Описание;
*   Минимальная версия NVDA;
*   Последняя протестированная версия NVDA.
